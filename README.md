Application that was created with purpose to get information about wished repository throught the link that you will provide.

----BEFORE STARTING!----
1.You are required to have your own git auth token!
2.Check link below to get the token:
https://docs.github.com/en/actions/concepts/security/github_token
3.After getting the token you have to run the command:
setx token "your_github_token_here"
4.After restart the terminal 
5.Copy repositery's link from the home page of the repository(Example: https://github.com/meinertvladyslav/GitHub_Analyzer-)
